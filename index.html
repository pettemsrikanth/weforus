    • Content Models
      • So for we assumed that HTML elements either belong to Inline or Block elements .
      • But now we have seven different content models .
      • Every HTML element belongs to one or more of these models .
      • 7 content models are

        1. Metadata

          Content that sets up the presentation or behavior of the rest of the content .


        2. Embedded

          Content that imports other resources into the document .


        3. Interactive

          Content sepcifically intended for user interaction .


        4. Heading

          Content that defines a section header .


        5. Phrasing

          Content that has a number of inline level elements .


        6. Flow

          Content that includes majority of HTML5 elements .


        7. Sectioning

          Content that defines the scope of headings,navigation,content and footers .

    • Drag and Drop API
      • The drag&drop feature used to grab an object and drag it to a different location .
      • The drag and drop API is event based .

        Need a little bit Knowledge in javascript , To understand Webstorage .

      • Basic syntax


        < script type="text/javascript">
        function allowDrop(ev){
        ev.preventDefault();
        }
        function drag(ev){
        ev.dataTransfer.setData("text",ev.target.id);
        }
        function drop(ev){
        ev.preventDefault();
        var data = ev.dataTransfer.getData("text");
        ev.target.appendChild(document.getElementById(data));
        }
        < /script>

        < div id="box" ondrop="drop(event)" ondragover="allowDrop(event)" style="border:1px solid white;width:336;height:300;">
